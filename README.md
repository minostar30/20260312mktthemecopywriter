# 월간 마케팅 테마 카피라이터

연도와 월을 선택하면 해당 시즌의 주요 마케팅 이슈를 보여주고, 그에 맞는 마케팅 테마를 제안하는 웹 앱입니다.

## 기능

- **연도/월 선택**: 2024~2028년, 1~12월 선택
- **시즌별 마케팅 이슈**: 선택한 달의 주요 마케팅 포인트 표시
- **테마 제안**: "테마 만들기" 버튼 클릭 시 새로운 테마 랜덤 생성
- **히스토리 저장**: 제안된 테마를 선택 월별로 저장 (로컬스토리지)

## 실행 방법

```bash
npm install
npm run dev
```

## 빌드 및 배포

```bash
npm run build
```

Vercel에 배포하려면 GitHub 저장소를 Vercel에 연결한 뒤 자동 배포됩니다.

## 예시 테마 (2026년)

| 월 | 테마 |
|---|---|
| 1월 | 올해 첫 피자는 도미노피자 |
| 3월 | 새학기, 새출발 도미노피자 |
| 4월 | 도미노 꽃이 피었습니다 |
